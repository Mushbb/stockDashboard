개인화 주식 대시보드: '오늘의 국장' 프로젝트 계획서
1. 프로젝트 개요
본 프로젝트는 국내 주식 시장에 관심 있는 투자자들을 위해 다양한 시장 정보를 한눈에 파악할 수 있는 개인화 대시보드 웹 애플리케이션을 개발하는 것을 목표로 합니다. 사용자는 자신이 원하는 정보 위젯을 직접 선택하고 배치하여, 자신만의 맞춤형 '오늘의 국장' 현황판을 구성할 수 있습니다.
2. 핵심 목표
* 정보의 집약: 분산된 시장 정보를 하나의 화면에 통합하여 제공함으로써 투자자의 정보 탐색 시간을 단축합니다.
* 개인화: 사용자가 직접 위젯을 선택하고 레이아웃을 구성하는 커스터마이징 기능을 제공하여 사용자 경험을 극대화합니다.
* 직관적 시각화: D3.js 등을 활용한 인터랙티브 차트를 통해 복잡한 데이터를 쉽고 직관적으로 이해할 수 있도록 돕습니다.
3. 주요 기능 (Features)
대시보드는 사용자가 선택하여 추가/제거할 수 있는 독립적인 **위젯(컴포넌트)**들의 집합으로 구성됩니다.
## Phase 1: 핵심 기능 구현 (MVP)
1단계에서는 사용자가 계정 없이도 시장의 전반적인 흐름을 즉시 파악할 수 있는 핵심적인 데이터 시각화 기능에 집중합니다.
* 주요 지수 현황
   * 코스피, 코스닥 지수의 실시간 등락률 및 차트와 투자자별(개인/외국인/기관) 순매수 현황을 제공합니다.
* 업종/테마 분석 트리맵
   * 시장 전체 뷰: 시장 전체를 조망하며 가장 강한 업종을 한눈에 파악할 수 있는 트리맵을 제공합니다.
   * 업종별 뷰: 특정 업종 내부의 주요 종목 등락 현황을 보여주는 트리맵을 제공합니다.
   * 두 뷰 모두 줌인/아웃(Zoomable) 기능을 탑재하여 사용자가 능동적으로 데이터를 탐색할 수 있도록 합니다.
* 시장 특징주 목록
   * 상한가, 거래량 급증, 신고가 등 시장의 주요 특징주 목록을 제공하여 시장의 특이 동향을 빠르게 파악할 수 있도록 합니다.
________________


## Phase 2: 개인화 및 상호작용 고도화
2단계는 사용자가 자신만의 대시보드를 만들 수 있도록 개인화 기능과 동적인 그리드 상호작용을 추가하는 데 집중합니다.
* 그리드 기반 대시보드 커스터마이징
   * React-Grid-Layout과 같은 라이브러리를 활용하여, 사용자가 1단계에서 구현된 그리드 내에서 위젯을 드래그 앤 드롭으로 이동시키고, 크기를 조절할 수 있는 완전한 인터랙티브 기능을 구현합니다.
   * 사용자가 위젯을 목록에서 선택하여 대시보드에 추가하거나 제거할 수 있는 기능을 제공합니다.
* 레이아웃 저장 및 복원
   * 사용자가 설정한 위젯의 종류, 위치, 크기 정보를 계정에 저장하고, 재방문 시 그대로 복원하는 기능을 추가합니다.
* 개인화 위젯 추가
   * 나의 관심 종목: 사용자가 직접 등록한 관심 종목의 시세를 추적하고 관리할 수 있는 개인화 위젯을 도입합니다.


4. 기술 아키텍처 및 스택
* 프레임워크: React.js
   * 독립적인 위젯 개발에 최적화된 **컴포넌트 기반 아키텍처(CBA)**를 활용합니다.
* 애플리케이션 구조: 단일 페이지 애플리케이션(SPA)
   * React Router를 사용해 페이지 새로고침 없는 부드러운 사용자 경험을 제공합니다.
* 데이터 시각화: D3.js
   * 줌 기능이 포함된 인터랙티브 트리맵 등 고도의 커스텀 시각화 구현에 사용합니다.
   * 필요에 따라 Chart.js 등 다른 라이브러리를 보조적으로 활용할 수 있습니다.
* 상태 관리: React Context API
   * AuthContext, UIContext 등 **역할 기반(Role-Based)**으로 Context를 분리하여, 예측 가능하고 유지보수가 용이한 상태 관리 구조를 설계합니다.
* 개발 환경: Vite
   * HMR(Hot Module Replacement) 및 프록시(Proxy) 기능을 활용하여 CORS 문제 없는 빠르고 효율적인 개발 환경을 구축합니다.
* 그리드 기반 레이아웃: React-Grid-Layout
* 백엔드 프레임워크: Spring Boot
   * 안정적이고 확장 가능한 RESTful API 서버를 구축하여 프론트엔드에 필요한 데이터를 제공합니다.
   * 주식 시장 데이터의 데일리 자동 갱신 및 데이터베이스 저장을 관리합니다.
5. 개발 단계 (MVP 접근법)
    ## Phase 1: 핵심 기능 구현 (완료된 MVP)
   * 개인화 기능이 제외된 핵심 위젯(주요 지수, 업종별 트리맵, 특징주) 개발에 집중합니다.
   * Spring Boot 백엔드를 통해 실시간 및 데일리 데이터를 자동 갱신하고 연동을 완료합니다.
   * 정적인 대시보드 레이아웃을 구성하고, 준비된 백엔드 API와의 데이터 연동을 완료합니다.
________________


## Phase 2: 아키텍처 고도화 및 기능 확장
기존 MVP의 기반 위에 대시보드의 확장성을 확보하고, 사용자에게 더 깊이 있는 분석 경험을 제공하는 단계입니다.
* 그리드 시스템 도입
   * 기존의 정적 레이아웃을 데스크탑 4x4, 모바일 2열 기준의 반응형 그리드 시스템으로 전환하여 향후 다양한 위젯을 수용할 수 있는 기반을 마련합니다.
* 신규 핵심 위젯 개발
   * 썬버스트 차트: 계층 구조 데이터를 원형으로 표현하는 새로운 시각화 위젯을 추가합니다.
   * 범용 리스트 컴포넌트 설계: 기존 '특징주 목록' 위젯을 재사용 가능한 컴포넌트로 리팩토링합니다. Props를 통해 컬럼, 데이터 등을 유연하게 설정할 수 있는 구조를 완성하여 향후 다양한 리스트 위젯(시총 상위 등)을 쉽게 추가할 수 있도록 합니다.
* 초기 개인화 기능 도입
   * 사용자 인증 시스템을 도입합니다.
   * 사용자가 등록한 종목을 추적하는 '나의 관심 종목' 위젯을 개발합니다.
________________


## Phase 3: 인터랙티브 대시보드 및 서비스 확장
사용자가 자신만의 대시보드를 완성하고, 더 넓은 범위의 데이터를 탐색할 수 있도록 서비스를 완성하는 단계입니다.
* 완전한 대시보드 커스터마이징
   * React-Grid-Layout 등의 라이브러리를 활용하여 사용자가 그리드 내에서 위젯을 직접 추가/제거하고, 드래그 앤 드롭으로 위치와 크기를 변경할 수 있는 완전한 인터랙티브 기능을 구현합니다.
   * 사용자가 설정한 레이아웃을 계정에 저장하고 복원하는 기능을 완성합니다.
* 지원 시장 및 위젯 확장
   * 미국 주식, 가상화폐 등 새로운 시장의 데이터를 볼 수 있는 위젯을 추가합니다.
   * 개별 종목 상세 차트, 기술적 분석 지표 등 더 다양한 전문 분석 위젯을 추가합니다.
* 지속적인 고도화
   * 늘어난 기능과 데이터에 맞춰 성능을 최적화하고, 사용자 피드백을 반영하여 UI/UX를 개선합니다.


6. 기대 효과
본 프로젝트를 통해 투자자는 복잡한 시장 정보를 자신의 입맛에 맞게 가공하여 빠르고 효율적으로 인사이트를 얻을 수 있는 강력한 웹 애플리케이션을 갖게 될 것입니다. 또한, React와 D3.js를 활용한 현대적인 웹 개발 기술을 집약적으로 적용하여 높은 완성도를 가진 포트폴리오를 확보할 수 있습니다.
